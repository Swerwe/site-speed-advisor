# Описание
Инструмент для анализа скорости загрузки веб-сайтов с использованием Puppeteer. Подходит для автоматизированного тестирования, мониторинга и сбора метрик производительности.
# Установка
```npm install```
# Использование
Запуск
```npm start https://aliexpress.ru```
# Сборка проекта
```npm run build```
# Технологии
- TypeScript
- Puppeteer
- ESLint + Prettier
- React
- MUI (Material UI)
